# 💰 Dashboard Financeiro

Dashboard financeiro pessoal moderno com tema escuro, gráficos interativos e controle completo de receitas e despesas.

## 📋 Sobre o projeto

Aplicação web para controle financeiro pessoal desenvolvida com HTML, CSS e JavaScript puro. Sem necessidade de backend ou banco de dados — todos os dados são salvos no navegador via localStorage.

## ✨ Funcionalidades

- 💳 **Saldo total** — cálculo automático de receitas menos despesas
- 📈 **Total de receitas** — soma e quantidade de entradas
- 📉 **Total de despesas** — soma e quantidade de saídas
- 🎯 **Taxa de economia** — percentual da renda economizada
- 📊 **Gráfico de barras** — receitas vs despesas dos últimos 6 meses
- 🥧 **Gráfico de pizza** — distribuição de gastos por categoria
- 📋 **Lista de transações** — com filtros por tipo e categoria
- ➕ **Adicionar transação** — receita ou despesa com categoria
- 🗑️ **Deletar transação** — remoção individual
- 📅 **Filtro por mês** — visualizar período específico
- 💾 **Persistência local** — dados salvos no navegador

## 🚀 Tecnologias

- **HTML5** — estrutura semântica
- **CSS3** — tema escuro, variáveis CSS, grid e flexbox
- **JavaScript** — vanilla JS, sem frameworks
- **Chart.js** — gráficos interativos
- **localStorage** — persistência de dados no navegador

## 🎨 Design

- Tema escuro moderno
- Layout responsivo para mobile e desktop
- Gráficos interativos com Chart.js
- Animações suaves nas transições
- Paleta de cores consistente com variáveis CSS

## ⚙️ Como usar

### Opção 1 — Direto no navegador
1. Baixe o arquivo `index.html`
2. Abra no navegador
3. Pronto!

### Opção 2 — GitHub Pages
1. Fork este repositório
2. Vá em **Settings → Pages**
3. Selecione a branch `main`
4. Acesse `seu-usuario.github.io/dashboard-financeiro`
